# 比较性投资辩论模板

> 适用于同赛道竞品对比辩论——双方各代表一个项目，基于同一组事实做不同解读。

## 适用场景

当用户要求比较两个同赛道竞品公司时，辩论比逐项分析更能暴露核心分歧。

## 设定规则

- **双方身份**：同机构同级别分析师，各自负责一个项目
- **资料基础**：双方都拿到对方的完整资料包——不是各执一词，而是基于同一组事实做不同解读
- **辩题格式**：赛道 + 路线分歧 + 估值对比。例如"具身数据赛道，仿真路线 vs 真机路线"
- **裁判角色**：中立，但有权查数据、挑事实错误、引导深入

## 核心分歧维度

经验表明，这类辩论中正反方会自然围绕以下五个维度交锋：

| 维度 | 典型正方论点 | 典型反方论点 |
|------|------|------|
| **技术路线** | 新路线是终局，旧路线是过渡 | 新旧并非替代关系，是互补关系 |
| **时间判断** | 颠覆比市场想的快 | 过渡期比对手想的久 |
| **估值逻辑** | 为稀缺性付溢价 | 为确定性付钱 |
| **风险认知** | 对手的结构性缺陷不可修复 | 己方的瑕疵是过程问题 |
| **客户证据** | 标杆客户的选择 = 行业方向 | 付钱的客户的选择 = 真实需求 |

## 辩论深度推进技巧

当辩论在立论层面僵持时，裁判应引导进入以下深度：

### 第一层 → 第二层：从"谁对谁错"到"谁对到什么程度"

- **不要问**："A 能替代 B 吗？"（是/否问题）
- **要问**："A 的份额从 30% 到 60% 需要多久？什么条件会加速/减速？"（程度问题）

### 第二层 → 第三层：从"替代"到"分工"

反方最有力的反驳框架是将"替代"重构为"分工"——证明两者不是零和博弈，而是互补关系。正方如需取胜，必须证明边界在向上移动。

### 第三层 → 第四层：从"当前"到"加速度"

正方最有力的论证是"替代不是线性的"——引用技术采纳的 S 曲线。一旦关键差距越过临界点，客户行为会从"试用"跳变为"主用"。

## 裁判实战技巧

1. **事实核查是武器**：双方都会选择性引用数据。裁判的价值是纠正偏差、补充上下文
2. **钱不说谎**：当技术路线之争无法仲裁时，看客户用钱投了哪边——收入/合同是最硬的数据
3. **暴露逻辑跳跃**：
   - 正方常见跳跃："标杆客户在试用 → 全行业即将采纳"
   - 反方常见跳跃："现在主流是 X → X 将长期是主流"
4. **框架图是收束工具**：每轮结束画 ASCII 图，将碎片化的论点收敛到结构化的分歧地图上

## 收敛时机

2-3 轮攻防通常足够暴露核心分歧。以下信号表明可以终止：
- 双方开始重复论点
- 核心分歧已清晰化为"对一个未知变量的不同判断"
- 裁判可以给出"双方都没有错，分歧取决于一个尚未被市场回答的问题"
